American Library Association. Guidelines for the Development and Promotion of Multilingual Collections and Services. Chicago: American Library Association. This guideline outlines professional standards for libraries that want to create multilingual collections and services. It covers best practices for collection development, community outreach, and ensuring equitable access to information for multilingual populations. This resource is helpful because it provides an official framework that librarians can follow when designing multilingual services.
